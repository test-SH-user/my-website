# My First Site

Personal portfolio website built with plain HTML and CSS.

## Style

- **Theme:** Dark background, light text
- **Design:** Modern and clean — minimal decoration, generous whitespace, clear typography
- **No frameworks:** Vanilla HTML and CSS only; no build tools, no JavaScript libraries unless absolutely necessary

## Code Conventions

- Keep HTML semantic (`<main>`, `<section>`, `<nav>`, `<article>`, etc.)
- CSS custom properties (variables) for colors, spacing, and fonts
- Mobile-first responsive design using flexbox/grid
- No inline styles

## Git Workflow

- **Commit messages:** Descriptive and specific — explain *what* changed and *why*, not just "update styles"
- **Commits:** Small and focused — one logical change per commit
- Example: `Add hero section with name, title, and call-to-action button` rather than `update index.html`
